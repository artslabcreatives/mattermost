üìã How It Works

  1. User visits /email_login?email=USER_EMAIL
  2. Frontend redirects to API endpoint: /api/v4/users/login/email_only
  3. Backend validates the email and creates a session
  4. Backend sets authentication cookies
  5. Backend returns HTML with auto-redirect to Mattermost home or specified channel
  6. User is logged in! ‚ú®
